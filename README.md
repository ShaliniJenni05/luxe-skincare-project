# Luxe Skincare Project
A full-stack inspired skincare e-commerce frontend.

## Features
* *Dynamic Shop:* Loads products from a local MySQL database with a fallback for static hosting.
* *Interactive UI:* Includes a welcome promotion modal and "Add to Cart" functionality.
* *Responsive Design:* Optimized for various screen sizes.

## Built With
* HTML5 / CSS3
* JavaScript (ES6)
* Node.js & MySQL (Local environment)
